# 문서 작성 스킬

프로젝트 문서(README, API docs, 가이드 등) 작성 시 활성화됩니다.

## 문서 종류별 가이드

### README.md
```markdown
# 프로젝트명

한 줄 설명

## 주요 기능
- 기능 1
- 기능 2

## 빠른 시작

### 사전 요구사항
- Python 3.11+
- PostgreSQL 15+

### 설치
(설치 명령어)

### 실행
(실행 명령어)

## 프로젝트 구조
(디렉토리 구조)

## 개발 가이드
(개발 시 알아야 할 것들)

## 라이선스
```

### API 문서
- 엔드포인트별: 메서드, 경로, 설명, 파라미터, 응답
- 에러 코드 목록
- 인증 방법
- 사용 예시 (curl 또는 코드)

### CHANGELOG
- Conventional Commits 기반으로 자동 생성 가능
- 형식: `## [버전] - YYYY-MM-DD`
- 카테고리: Added, Changed, Fixed, Removed

## 작성 원칙
- 코드를 모르는 사람도 이해할 수 있게
- 예시를 반드시 포함
- 최신 상태 유지 (코드 변경 시 문서도 업데이트)
- 과도한 문서화보다 핵심만 간결하게
