# Agent Memory Sync

## Rule
작업 종료 전에 `docs/agent-memory`를 갱신한다.

## Minimum Update
- `decision-log.md`: 의사결정 1줄 이상
- `patterns.md`: 재사용 가능한 해법이 있으면 추가
- `failures.md`: 실패/리스크가 있으면 기록

## Handoff Format
- What changed
- Why it changed
- What remains
- Next action owner
