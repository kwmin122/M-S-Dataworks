# KiraBot E2E 테스트 프롬프트 (크롬 확장 Claude용)

> 아래 프롬프트를 크롬 확장 Claude에 붙여넣으세요.
> 사이트: https://m-s-solutions-production.up.railway.app

---

## 프롬프트 (복사용)

```
당신은 QA 테스터입니다. 아래 사이트의 모든 기능을 순서대로 테스트하고, 각 항목의 결과를 PASS/FAIL로 기록해주세요.

사이트: https://m-s-solutions-production.up.railway.app

## 테스트 진행 규칙
- 각 테스트 항목을 순서대로 실행
- 스크린샷이 필요하면 현재 화면 캡처
- FAIL인 경우 에러 메시지나 증상을 상세 기록
- 모든 테스트 완료 후 결과 요약표 작성

---

## Phase 1: 랜딩페이지 (로그인 전)

### T1.1 페이지 로딩
- 사이트 접속 → 랜딩페이지가 정상 표시되는지 확인
- 체크: Hero 섹션, "시작하기" 버튼, "공고 알림 설정" 버튼 표시

### T1.2 섹션 순서 확인
- 스크롤하면서 순서 확인:
  1. Hero
  2. HowItWorks (3단계 설명)
  3. Solutions (4개 산업 카드)
  4. Features (기능 소개)
  5. Pricing (Free / Pro 플랜)
- 체크: 위 순서대로 표시되는지

### T1.3 네비게이션
- 상단 Navbar 메뉴 클릭 → 해당 섹션으로 스크롤
- "로그인" 버튼 클릭 → 로그인 모달 표시
- 체크: 스크롤 이동, 모달 표시 정상

### T1.4 Pricing 섹션
- Free 플랜: 가격 "0원/월", "시작하기" 버튼
- Pro 플랜: 가격 "99,000원/월", "시작하기" 버튼
- Pro "시작하기" 클릭 → 로그인 모달 표시 (미로그인 상태)
- 체크: 가격 표시, 버튼 동작

### T1.5 Footer 링크
- 하단 Footer → "이용약관", "개인정보처리방침" 클릭
- 체크: /terms, /privacy 페이지 정상 표시

---

## Phase 2: 로그인

### T2.1 Google 로그인
- 로그인 버튼 클릭 → 로그인 모달 → "Google로 로그인" 클릭
- Google 계정 선택 → 로그인 완료
- 체크: 로그인 후 /chat 페이지로 이동, 상단에 사용자 이름/아바타 표시

### T2.2 보호된 라우트 접근
- 로그인 전에 직접 /chat 접속 시도 → 랜딩페이지로 리다이렉트
- 로그인 후 /chat 접속 → 정상 표시
- 체크: 미인증 시 리다이렉트, 인증 후 접근 정상

---

## Phase 3: 채팅 - 문서 분석 플로우

### T3.1 새 대화 시작
- /chat 페이지 → 환영 메시지 표시
- "문서 분석" 또는 관련 버튼 클릭
- 체크: 환영 메시지, 기능 선택 버튼 표시

### T3.2 회사 문서 업로드 (선택)
- "회사 문서 등록" 선택 시 파일 업로드 UI 표시
- PDF, DOCX, HWP 파일 중 하나 업로드
- 체크: 업로드 진행, 완료 메시지

### T3.3 분석 대상 문서 업로드
- 입찰공고/RFP 문서(PDF 또는 HWP) 업로드
- 체크: 업로드 후 "분석 중..." 상태 표시

### T3.4 분석 결과 확인
- 분석 완료 → 자격요건 추출 결과 표시
- GO/NO-GO 판정 결과
- 각 자격요건별 PASS/FAIL/주의 표시
- 체크: 분석 결과 카드, 자격요건 목록, GO/NO-GO 판정

### T3.5 문서 Q&A 채팅
- 분석 완료 후 자유 질문 입력 (예: "이 공고의 납품 기한은?")
- 체크: 답변 표시, 참조 페이지 번호 포함

### T3.6 컨텍스트 패널
- 우측 컨텍스트 패널에 업로드된 문서 미리보기
- PDF면 iframe으로 표시
- 체크: 문서 미리보기 정상, 탭 전환

---

## Phase 4: 채팅 - 공고 검색 플로우

### T4.1 공고 검색 시작
- "공고 검색" 선택 → 검색 폼 표시
- 체크: 키워드, 업무구분, 기간, 지역 필드 표시

### T4.2 검색 실행
- 키워드 "소프트웨어" 입력 → 검색
- 체크: 검색 결과 카드 목록 표시 (공고명, 발주기관, 지역, 마감일, 추정가)

### T4.3 페이지네이션
- 검색 결과가 많을 경우 "더 보기" 또는 페이지 이동
- 체크: 추가 결과 로딩

### T4.4 공고 상세 분석
- 검색 결과에서 특정 공고 "분석하기" 클릭
- 첨부파일 자동 다운로드 → 분석 실행
- 체크: 분석 진행, 결과 표시 (GO/NO-GO)
- 참고: 첨부파일 없는 공고는 수동 업로드 안내 메시지 표시될 수 있음

### T4.5 일괄 평가
- 여러 공고 선택 → "일괄 평가" 실행
- 체크: 각 공고별 평가 결과 목록

---

## Phase 5: 사이드바

### T5.1 대화 목록
- 왼쪽 사이드바에 대화 목록 표시
- 체크: 이전 대화 클릭 시 해당 대화 로딩

### T5.2 새 대화
- "새 대화" 버튼 클릭 → 새 대화 시작
- 체크: 환영 메시지로 초기화

### T5.3 대화 이름 변경
- 대화 항목 우클릭 또는 메뉴 → 이름 변경
- 체크: 이름 변경 후 반영

### T5.4 대화 삭제
- 대화 삭제 → 확인 후 삭제
- 체크: 목록에서 제거

---

## Phase 6: 설정 페이지

### T6.1 설정 진입
- 왼쪽 사이드바 또는 프로필 메뉴 → 설정
- 체크: /settings/general 페이지 표시

### T6.2 일반 설정 (General)
- 사용자 이름, 이메일 표시
- 체크: 로그인한 사용자 정보 정확

### T6.3 회사 정보 (Company)
- /settings/company 이동
- 회사명, 업종, 사업자번호, 인증서, 지역, 인원, 매출, 주요 경험 필드
- 문서 업로드 (회사소개서 등) → AI 자동 추출
- "저장" 클릭
- 체크: 폼 필드 입력/수정, 문서 업로드, AI 추출, 저장 정상

### T6.4 사용량 (Usage)
- /settings/usage 이동
- 대시보드 카드: 신규매칭, 마감임박, GO건수, 총분석건수
- 체크: 숫자 표시 (초기에는 0일 수 있음)

### T6.5 구독 관리 (Subscription)
- /settings/subscription 이동
- 현재 구독 상태 표시 (Free면 "구독 정보가 없습니다")
- 체크: 구독 상태 표시

### T6.6 계정 (Account)
- /settings/account 이동
- 로그아웃 버튼 표시
- 체크: 로그아웃 버튼 존재 (클릭하지 마세요)

---

## Phase 7: 알림 설정

### T7.1 알림 설정 진입
- /settings/alerts 이동
- 체크: 알림 설정 페이지 표시

### T7.2 알림 규칙 설정
- 전체 활성화 토글 ON
- 이메일 입력 (예: test@test.com)
- 알림 주기 선택 (매일 1회)
- "규칙 추가" → 키워드: "소프트웨어", 카테고리 선택, 지역 선택
- 체크: 규칙 폼 정상, 필드 입력 가능

### T7.3 알림 저장
- "저장" 클릭
- 체크: 저장 성공 메시지

### T7.4 알림 즉시 발송 테스트
- "지금 발송" 버튼이 있으면 클릭
- 체크: 발송 처리 (이메일 수신은 실제 이메일로 확인 필요)

---

## Phase 8: 발주예측

### T8.1 예측 페이지 진입
- /forecast 이동
- 체크: 인기 발주기관 목록 표시

### T8.2 기관 선택
- 인기 기관 중 하나 클릭 (예: "한국도로공사")
- 체크: 월별 발주 패턴 차트 표시

### T8.3 차트 확인
- 건수/금액 토글 전환
- 카테고리별 분포 차트
- 체크: 차트 정상 렌더링, 토글 동작

### T8.4 기관 상세 정보
- 집중 발주 시기
- 발주 트렌드 (증가/감소)
- 평균 발주 규모
- 주력 카테고리
- 체크: 인사이트 카드 표시

### T8.5 최근 공고 목록
- 해당 기관의 최근 공고 20건 목록
- 체크: 공고 목록 표시 (제목, 날짜, 금액)

### T8.6 사전공개 (발주계획)
- 발주계획 목록이 있으면 표시
- 체크: 목록 표시 (없으면 "없음" 메시지)

---

## Phase 9: 결제 플로우 (확인만)

### T9.1 결제 모달
- 랜딩페이지 → Pricing → Pro "시작하기" (로그인 상태)
- 체크: PaymentModal 표시 (플랜: Pro, 금액: 99,000원/월, "카드 등록하고 구독 시작" 버튼)
- 주의: 실제 결제하지 마세요! 모달 표시만 확인 후 닫기

## Phase 10: 로그아웃

### T10.1 로그아웃
- 프로필 메뉴 또는 설정 > 계정 → 로그아웃
- 체크: 랜딩페이지로 이동, 로그인 상태 초기화
```
